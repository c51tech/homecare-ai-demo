# HomeCare AI — TRL 6 Prototype

AI 기반 방문진료 전주기 통합지원 시스템 프로토타입

## 포함 데모

- **UI 프로토타입**: 7화면 전체 플로우 (방문일정 → AI 브리핑 → CGA-FI 문진 → 활력징후 → K-MMSE/PHQ-9 → SOAP 생성 → 방문완료)
- **AI 라이브 데모**: Claude API 실시간 연동, 환자 브리핑 + SOAP 노트 스트리밍 생성

## 실행

```bash
npm install
npm run dev
```

## 배포

Vercel에 이 레포를 연결하면 자동 배포됩니다.
